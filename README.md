### **Step-by-Step Installation**

#### **Step 1: Clone the Repository**

#### **Step 2: Set Up Python Environment**
```bash

python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate

# Upgrade pip
pip install --upgrade pip
```

#### **Step 3: Install Dependencies**

##### **For Current Version (Recommended)**
```bash
# Install Python dependencies
pip install -r rag_api/requirements_simple.txt

# Verify installation

```

##### **For LlamaIndex Enhanced Version**
```bash

# Additional LlamaIndex dependencies

# Verify LlamaIndex installation
python -c "import llama_index; print('‚úÖ LlamaIndex installed successfully!')"
```

#### **Step 4: Run the Application**

##### **Option A: Direct Python Execution**
```bash
# Navigate to API directory
cd rag_api

# Run the application
python api.py

# Or run the optimized version
python main_optimized.py
```

##### **Option B: Using Uvicorn (Recommended)**
```bash

cd rag_api

# Run with uvicorn
uvicorn api:app --host 0.0.0.0 --port 8000 --reload

# Or for production
uvicorn api:app --host 0.0.0.0 --port 8000
```

##### **Option C: Windows Batch File (Windows Only)**
```bash

rag_api\LAUNCH_ENHANCED.bat
```

##### **Option D: LlamaIndex Enhanced Version (Advanced)**
```bash

pip install -r rag_api/requirements_simple.txt

# Run LlamaIndex version
cd rag_api
python api_llamaindex.py

# Or with uvicorn

```
